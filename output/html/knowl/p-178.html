Addition and subtraction between arrays is done elementwise. For example, we can add the two arrays [1, 6, 3] and [5, 11, 2] as follows.

\begin{align*} [1, 6, 3] + [5, 11, 2] \amp = [\left(1 + 5\right), \left(6 + 11\right), \left(3 + 2\right)]\\ \amp = [6, 17, 5] \end{align*}

Subtraction between those two arrays is done similarly.

\begin{align*} [1, 6, 3] - [5, 11, 2] \amp = \left[\left(1 - 5\right), \left(6 - 11\right), \left(3 - 2\right)\right]\\ \amp = [-4, -5, 1] \end{align*}
