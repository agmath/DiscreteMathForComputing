The Matrix-Tree Theorem. The matrix tree theorem uses matrix determinants to find the number of spanning trees in a graph. In practice, this could help you find - for instance - the number of routes a postal worker could take to deliver mail to every house in a neighborhood. You should talk about what graphs and spanning trees are, how to encode a graph in a matrix, and then show how to find the number of spanning trees on a small graph.
