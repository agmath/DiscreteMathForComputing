I'm envisioning a series of interactive problems to make up the bulk of this section. Sample exercises are described below. I'll write dynamic versions into MyOpenMath soon and embed them into the text.

  1. A single character password is to be chosen over an approved alphabet of characters. How many unique passwords are possible?

  2. A single character password is to be chosen over an approved alphabet of characters or a selected set of symbols. How many unique passwords are possible?

  3. A single character password is to be chosen over an approved alphabet of characters, selected set of symbols, or the digits 0 - 9. How many unique passwords are possible?

  4. A two-character password is to be chosen...

  5. etc.
